cv_bfmatcher

To match descriptors

Calling Sequence

matches=cv_bfmatcher(des1,des1)

Arguments

des1

Descriptors of an image

des1

Descriptors of another Image

matches

matches is a list of DMatch objects.

Description

Match Descriptors

bf_matcher is used to match descriptors from 2 differnt images, It return DMatch object which has properties like distance to know good matches and bad matches

Examples

matches=cv_bfmatcher(des1,des2)
